DESCRIPCIÓN^^^^^^^^^^^^^^^^^^^^^^^^^^
Este repositorio cuenta con un proyecto de Java al cual se le dan las especificaciones de la casa.^



INSTRUCCIONES^^^^^^^^^^^^^^^^^^^^^^^^

El programa solicita los datos de una casa de la siguiente forma: HALL, COCINA, SALÓN, HABITACIONES* y BAÑOS*.

*(Número y datos de cada uno).

DATOS QUE HAY QUE INTRODUCIR PARA CADA CLASE DE ESTANCIA:
***Todas las estancias**** cuentan con un nombre , una superficie en metros cuadrados (Nº Entero), un número de puertas (Nº Entero)
y un número de ventanas (Nº Entero). 

Del *HALL* se pide también si tiene puerta blindada (S/N).

De cada *COCINA* se pide también si incluye tendedero (S/N) y si tiene, su superficie en metros cuadrados (Número Entero).

De cada *SALÓN* se pide también el número de tomas de corriente que tiene (Nº Entero), si incluye terraza (S/N)
y, en caso de que tenga, su superficie en metros cuadrados(Nº Entero).

De cada *HABITACIÓN* , se pide también el número de puertas del armario empotrado (Nº Entero) 
y si tiene acceso directo a un baño (S/N).

De cada *BAÑO* se pide si tiene bañera o ducha (B/D).

El programa devuelve los datos de todas las estancias y la superficie total de la casa en metros cuadrados.
